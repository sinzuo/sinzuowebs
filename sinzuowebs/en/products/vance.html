点播云

点播云 (VANCE) 是汇聚芯卓信息科技存储、音视频转码和融合 CDN 等强大、稳定的基础服务，专注于视频领域为客户提供视频上传、存储、转码、播放器、分发加速和统计分析等服务的、专业可靠的一站式视频点播 PaaS 平台。

产品架构

核心功能

视频上传 SDK

支持本地文件上传、API 上传、并提供 Android、iOS 端功能丰富的上传 SDK

统计分析

提供用量统计、运营统计和质量统计等多维度统计分析；开放运营数据，助力客户自定义统计分析

H. 265 转码

支持 H.265 转码压缩技术，可节省存储、带宽达 10%－30%

静态转码

根据转码预设，视频上传完毕后即转码，简化转码流程，满足各种使用场景

Web 播放器

Web 播放器，适配最流行的设备和浏览器，支持 HLS 自适应多码率

HLS自适应多码率

支持 HLS 自适应多码率转码和播放，根据网络带宽实时切换到最佳码率，保障流畅视听体验

iOS/Android 播放器

支持播放 H.265、HLS 自适应多码率，对标主流视频门户网站，首屏秒开

动态转码

按需转码，边转边播，节约存储空间

产品优势

安全可靠

依托芯卓信息科技可靠的云存储服务，保障数据安全，同时提供鉴权、防盗链等多种安全防护。

节约成本

按需使用和付费，利用 H.265 转码技术压缩视频节省存储、带宽高达 10％ - 30%。

专业音频处理

专注于音视频领域，提供音视频转码（支持 H.265 转码）、视频分发、全平台播放及主动开放式数据分析等功能。

极致视听体验

多 CDN 融合方案，保证视频加速无盲区；支持 HTTPS 加速，HLS 多码流自适应；视频流畅率达 95%+，无缓冲比 > 95%。
